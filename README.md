# ばぶちゃんCodexガイド

Codexでできることを、初心者向けのやさしい言葉で説明するシンプルなWebアプリです。

## 使い方

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開くと表示されます。

## サムネイル（OGP）について

`index.html` に OGP タグを設定しています。Discord などで画像付きカードを出すには、
`og:image` を公開URL（`https://...`）にしてください。


## Safariで反映されない時

- 画面上部に `反映確認` と `Build: v2026.02.21-3` が見えるか確認してください。
- ページ内の `Safari向け 反映リロード` ボタンを押すとURLにタイムスタンプを付けて再読込します。
- GitHub Pages側は、再デプロイ後に数分のキャッシュ遅延が出ることがあります。
